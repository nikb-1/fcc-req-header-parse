Header Request Parser Microservice API


This API is a Free Code Camp - Back End Development Certification project.

It can be used by opening the url path below or via a GET request to the same url path.

The API returns a JSON string with 3 properties; "ipaddress", "language", "software", all of which are derived from the request header

Example Usage

https://fcc-req-header-parse-nikburazor.herokuapp.com/api/whoami/

Example Output

{"ipaddress":"49.2.82.143","language":"en-US","software":"Windows NT 10.0; WOW64; rv:47.0"}
